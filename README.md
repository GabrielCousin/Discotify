# Discotify

Export your Discogs library to Spotify


## Install

```bash
$ yarn
$ # npm install
$ cp src/config/settings.js.dist src/config/settings.js
$ # edit with your own tokens
```


## Development

```bash
$ yarn start
$ # npm run start
```


## Production

```bash
$ yarn build
$ # npm run build
```


## Test

```bash
$ yarn test
$ # npm run test
```
